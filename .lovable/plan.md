## Method

For each substitution, edit each file with case-aware replacements (`catalogue`/`Catalogue`/`CATALOGUE`) using targeted line edits, skipping any line that:
- contains an `http://` or `https://` URL with the British word inside the URL, or
- is one of the proper-noun lines listed under "Hard exclusions".

For `htaExporter.ts` specifically: check whether the AID-RT export spec fixes "Multi-centre" as the column key. If yes, leave the literal value untouched and only adjust surrounding prose/comments; if it's free-form, change to "Multi-center".
